选择排序、冒泡排序

一月 15, 2020 发布在 算法学习笔记, 算法

选择排序

时间复杂度

O(n²)

原理

选择排序是最简单的排序算法，原理也简单直观：遍历列表n次，每次找到并移除列表中最小的元素，放入一个新的列表中。最后这个新的列表就是一个有序列表。

由于一共需要遍历n次，每次需要检查的元素个数是递减的，所以总的时间复杂度为：n + (n - 1) + （n - 2）+ …… + 1。化简后可知时间复杂度为O(n²)。

Python实现

def findSmallest(arr):
    smallest = arr[0]
    smallest_index = 0
    for i in range(1, len(arr)):
        if arr[i] < smallest:
            smallest = arr[i]
            smallest_index = i
    return smallest_index

def selectionSort(arr):
    newArr = []
    for i in range(len(arr)):
        smallest = findSmallest(arr)
        newArr.append(arr.pop(smallest))
    return newArr

冒泡排序

时间复杂度

O(n²)

原理

冒泡排序是一个简单而且经典的算法。其原理是遍历列表中的所有元素，将每个元素和它后面的一个元素进行比较，如果比排在后面的元素大则交换位置。遍历一直重复n-1次（要使列表变为有序最多需要遍历列表n-1次）。

Python实现

def bubbleSort(arr):
    for i in range(0, len(arr) - 1):
        for j in range(0, len(arr) - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

简单优化

当列表已经有序时就不需要继续遍历列表，可以创建一个flag变量来表明列表是否已经有序。

def bubbleSort(arr):
    change = false
    for i in range(0, len(arr) - 1):
        for j in range(0, len(arr) - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                change = true
        if change == false:
            return arr
    return arr
