二分查找

一月 8, 2020 发布在 算法学习笔记, 算法

功能

二分查找的功能是在一个有序的元素列表中查找一个特定的元素。

时间复杂度

O(log n)

原理

比方说你和朋友在玩猜数游戏，朋友先想好一个1到100的数，由你来猜这个数是多少。每猜一次朋友都会告诉你你猜的数是大了还是小了。要解决这个问题有两个办法。

第一个办法是从1开始猜，一直猜到100。但如果对方想的数是100，那我们就需要猜100次才能猜对。这种方法的效率比较低。这种方法相当于遍历整个列表，进行一一比较。

第二个办法是先猜50，如果小了，那么小于等于50的数字就都被排除了。每次都猜范围内的中间数。这样我们每次都能排除掉一半的数字，效率提高了很多。这就是二分查找。

Python实现

def binarySearch(list, item):
    low = 0
    high = len(list) - 1
    while low <= high:
        mid = (low + high) / 2
        guess = list[mid]
        if guess == item:
            return mid
        if guess > item:
            high = mid - 1
        else:
            low = mid + 1
    return None

ls = [1, 2, 3, 4, 5]
print(binarySearch(ls, 2))
